Instant Search v1.0.2
Just like googles instant search feature I’ve made one for zencart. Instant search is a new search enhancement that shows results as you type. 

Tried and tested and works on all major browsers including smart phones.

Installation
None of the core Zencart files are affected, so in theory it shouldn't affect any future upgrades. Tested with ZC1.5.5f and ZC1.5.6 using PHP7.1.11 and php7.2.4 as well, subjected to minor sql injection attacks without failing.

Change the template from YOUR_TEMPLATE to your template name you are using!!! Then copy the files in the INSTALL folder to the equivalent places in your site. They already have the correct folder structure so a single drag and drop into the root directory for your store will put the files in the correct locations.

  • searches.php
  • includes/templates/YOUR_TEMPLATE/css/stylesheet_instantSearch.css
  • includes/templates/YOUR_TEMPLATE/jscript/jscript_instantSearch.js

Intent Search is based on jQuery and needs it installed to work correctly. ZC1.5.5x comes with jQuery setup in the responsive template. You need to make sure jQuery is loaded before any script requiring it. Instant Search was tested using jQuery v3.2.1 and v1.12.4

 

How It Works

  1. When the page loads up jscript_instantSearch.js adds a keyup listeners to the search boxes.
  2. When the user starts to key in a few letters jscript_instantSearch.js sends the data to searches.php.
  3. searches.php uses sql to gather matching search results.
  4. searches.php sends these results back to jscript_instantSearch.js.
  5. jscript_instantSearch.js then creates the instant search box with the results on the web page.
  • Instant search uses the jQuery JavaScript Library. Check out http://jquery.com/
  • You can change the style and layout of the instant search via stylesheet_instantSearch.css.
